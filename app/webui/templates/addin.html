Deployment im M365 Admin Center

  1. admin.microsoft.com → Einstellungen → Integrierte Apps
  2. Apps hochladen → Office-Add-In
  3. Link zur Manifestdatei → obige URL einfügen → Validieren
  4. Benutzer/Gruppen zuweisen (z. B. EXO Signature Gateway – Enabled Mailboxes)
  5. Das Add-in erscheint in Outlook nach bis zu 24 Stunden

Plattformen: Outlook Desktop (Windows & Mac), OWA, Outlook Web  ·  Auth: Nutzer werden beim ersten Öffnen zur Gateway-Anmeldung geleitet  ·  Proxy: Azure Application Proxy Pre-Authentication auf Passthrough setzen

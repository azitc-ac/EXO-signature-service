ACME ACME Challenge Reply — Versandmethode — Graph API (Standard) oder Direktversand zum Castle MX

Steuert wie die ACME-Challenge-Antwort an acme@castle.cloud gesendet wird. Graph API läuft über Exchange (Double-Hop via Transport-Connector). Direktversand verbindet direkt zu castle.cloud MX auf Port 25 — umgeht Exchange komplett, aber nur nutzbar wenn Port 25 outbound nicht geblockt ist.

Lab Exchange Header Observatory — testet welche Header Exchange bei ACME-Antwortmails hinzufügt

Schickt eine Test-ACME-Reply via Graph API sendMail durch Exchange. Das Gateway fängt die Nachricht auf dem Outbound-Connector ab und speichert den exakten Raw-MIME — so sieht man welche Header Exchange hinzufügt bzw. verändert.
Optional: RemoteDomain für castle.cloud setzen, dann erneut testen und die Ergebnisse vergleichen.
